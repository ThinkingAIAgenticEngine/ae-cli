# te-engage +approver-list

> **前置条件:** 阅读 [`../../te-shared/SKILL.md`](../../te-shared/SKILL.md)

查询项目审批人列表。

映射命令: `te-cli te-engage +approver-list`

## Flags

| Flag | 类型 | 必填 | 说明 |
|------|------|------|------|
| `--project-id` / `-p` | number | 是 | 项目 ID |

## 示例

```bash
te-cli te-engage +approver-list --project-id 1
```
